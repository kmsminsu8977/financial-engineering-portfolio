## Category Scope

이 카테고리는 금융 데이터를 수집·정제·분석·시각화하는 전 과정을 다룹니다. 모델 성능뿐 아니라 데이터 파이프라인의 안정성, 재사용성, 확장성을 함께 고려합니다.

주요 주제 예시는 다음과 같습니다.

- 금융 뉴스 NLP
- 시장 데이터 분석 및 시각화
- 데이터 수집/적재 파이프라인
- 금융 데이터 마이닝 및 예측 실험
- 리서치 자동화 워크플로우

## Intended Portfolio Output

향후 이 카테고리의 각 프로젝트는 아래 요소로 구성할 예정입니다.

- 데이터 출처 및 수집 방식
- 정제/가공 로직
- 분석 또는 모델링 방법
- 결과 요약(지표, 시각화, 인사이트)
- 운영 관점(자동화, 유지보수, 한계)
- 관련 repository, Notion, slides 링크
